# Dispatch Warehouse Management System

This project is a **Dispatch Warehouse Management System (WMS)** built using Angular and Angular Material.

The goal of this system is to manage warehouse dispatch operations such as inventory management, order dispatch, and shipment tracking.

---

## Tech Stack

* Angular 21
* Angular Material
* TypeScript
* RxJS
* SCSS

---

## Project Setup

Install dependencies:

```bash
npm install
```

Start development server:

```bash
ng serve
```

Application will run at:

```text
http://localhost:4200
```

---

## Available Scripts
